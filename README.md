# clean-job-ads

A clean list of job ads

A button to add job ad that shows modal and then creates the job ad and refreshes the job ads

The job ad shows modal of that job ad

## Setup

```sh

cd clean-job-ads

```


This will run a dev server with auto reloading and an instant preview.
